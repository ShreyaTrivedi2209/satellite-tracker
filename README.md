# Satellite Tracker

This repository contains the Satellite Tracker app in `satellite-tracker/`.

```bash
cd satellite-tracker
npm install
python -m pip install -r python/requirements.txt
```

Start the backend in one terminal:

```bash
npm run api
```

Start the frontend in another terminal:

```bash
npm run dev
```

Then open the local URL printed by Vite, usually:

```text
http://127.0.0.1:5173/
```

See `satellite-tracker/README.md` for the full setup, validation, and GitHub push instructions.
